# е…ідәҺжҲ‘жғіеӨҮд»Ҫ blog иҝҷд»¶дәӢ

жҲ‘иҰҒжҖҺд№ҲеӨҮд»Ҫblogж–Үд»¶пјҹпјҹпјҹ

иҝҷеҮ еӨ©еңЁжғіпјҢдёҮдёҖйӮЈеӨ©жҲ‘жғіеңЁеҲ«зҡ„з”өи„‘дёҠеҶҷеҚҡе®ўпјҢжҲ–иҖ…жҳҜжҲ‘з”өи„‘зӘҒз„¶жҠҪйЈҺзЎ¬зӣҳеқҸдәҶпјҢжҲ‘зҡ„еҚҡе®ўж•°жҚ®еІӮдёҚжҳҜжІЎдәҶпјҹпјҹпјҹ

еҸҲжҲ–иҖ…жҲ‘д»Җд№Ҳж—¶еҖҷдёҚе°ҸеҝғеҲ й”ҷдәҶж–Үд»¶пјҢеІӮдёҚжҳҜеҮәеӨ§й—®йўҳпјҹпјҹпјҹ

еёҰзқҖиҝҷдёӘй—®йўҳпјҢжҲ‘дёҖејҖе§ӢжҳҜжүҫдәҶеҮ дёӘ PCз«Ҝ иҮӘеҠЁжң¬ең°еӨҮд»Ҫзҡ„зЁӢеәҸпјҢдҪҶж•ҲжһңдёҚеӨӘзҗҶжғігҖӮ

е®ҡж—¶еӨҮд»ҪпјҡдёҚиғҪдҝқиҜҒдҪ еңЁи®ҫзҪ®е®ҡж—¶зҡ„иҝҷж®өж—¶й—ҙеҶ…иҝӣиЎҢ hexo d еӨҡе°‘ж¬ЎпјҢеҰӮжһңе®ҡж—¶еӨҮд»Ҫзҡ„ж—¶й—ҙи®ҫзҪ®иҝҮзҹӯпјҢзҗҶи®әдёҠеҜ№зЎ¬зӣҳзҡ„иҜ»еҶҷдјҡжӢ–ж…ўе…¶д»–зЁӢеәҸзҡ„иҝҗиЎҢгҖӮ

жүҖд»ҘеңЁжғіпјҢеҰӮжһңжңүдёҖдёӘзЁӢеәҸеҸҜд»Ҙзӣ‘жҺ§жҲ‘ blog дёӢзҡ„ж–Үд»¶еҸҳеҢ–пјҢжҜҸеҸҳеҢ–дёҖж¬Ўе°ұиҝӣиЎҢи®°еҪ•дёҖж¬ЎпјҢиҝӣиЎҢеўһйҮҸеӨҮд»ҪдёҖж¬ЎпјҢиҝҷжҳҜжҲ‘жғіеҲ°жңҖе®ҢзҫҺзҡ„еҠһжі•пјҢеёҰзқҖиҝҷдёӘжғіжі•пјҢжҲ‘дјјд№ҺжүҫеҲ°дәҶдёҖдёӘж–№жі•гҖӮ

жҢҒз»ӯйӣҶжҲҗжңҚеҠЎ

жӯӨеӨ„еј•з”ЁзҷҫеәҰзҷҫз§‘пјҡжҢҒз»ӯйӣҶжҲҗжҳҜдёҖз§ҚиҪҜд»¶ејҖеҸ‘е®һи·өпјҢеҚіеӣўйҳҹејҖеҸ‘жҲҗе‘ҳз»ҸеёёйӣҶжҲҗд»–д»¬зҡ„е·ҘдҪңпјҢйҖҡеёёжҜҸдёӘжҲҗе‘ҳжҜҸеӨ©иҮіе°‘йӣҶжҲҗдёҖж¬ЎпјҢд№ҹе°ұж„Ҹе‘ізқҖжҜҸеӨ©еҸҜиғҪдјҡеҸ‘з”ҹеӨҡж¬ЎйӣҶжҲҗгҖӮжҜҸж¬ЎйӣҶжҲҗйғҪйҖҡиҝҮиҮӘеҠЁеҢ–зҡ„жһ„е»әпјҲеҢ…жӢ¬зј–иҜ‘пјҢеҸ‘еёғпјҢиҮӘеҠЁеҢ–жөӢиҜ•пјүжқҘйӘҢиҜҒпјҢд»ҺиҖҢе°Ҫж—©ең°еҸ‘зҺ°йӣҶжҲҗй”ҷиҜҜгҖӮ

з®ҖеҚ•жқҘиҜҙпјҢе°ұжҳҜиҝҷдёӘжңҚеҠЎз»‘е®ҡдәҶ GitHub пјҢжҲ‘д»¬жҜҸжҸҗдәӨдёҖж¬Ўд»Јз ҒеҲ° GitHub пјҢе®ғйғҪдјҡжҠ“еҸ–дёҖж¬ЎжҲ‘д»¬зҡ„д»“еә“пјҢжЈҖжҹҘж–°д»Јз ҒпјҢз„¶еҗҺжҸҗдҫӣдёҖдёӘиҝҗиЎҢзҺҜеўғпјҢжү§иЎҢжөӢиҜ•пјҢе®ҢжҲҗжһ„е»әпјҢиҝҳиғҪйғЁзҪІеҲ°жңҚеҠЎеҷЁгҖӮ

з„¶еҗҺжҲ‘иҝҷж¬ЎеңЁеҮҶеӨҮжҸҗдәӨж–Үд»¶еҲ°жҲ‘ж–°е»әзҡ„ hexo-backups д»“еә“зҡ„ж—¶еҖҷпјҢеҮәй”ҷдәҶпјҢе°ұи®°еҪ•дёҖдёӢй”ҷиҜҜгҖӮ

# git pash origin master еӨұиҙҘ

з¬¬дёҖж¬ЎйҒҮи§ҒиҝҷдёӘй—®йўҳпјҢжҲ‘иҝҳд»ҘдёәжҳҜ .gitignore ж–Үд»¶зҡ„ .DS_Store ж–Үд»¶дёҚеӯҳеңЁпјҹз„¶еҗҺжҲ‘еҲ жҺүиҝҷдёӘж–Үд»¶пјҢеҘҪеғҸиҝҳжҳҜз»§з»ӯжҠҘдёҖж ·зҡ„й”ҷиҜҜпјҢз„¶еҗҺжҺ’йҷӨиҝҷдёӘеҸҜиғҪгҖӮ

# ж–°е»әж–Үд»¶жөӢиҜ•

з„¶еҗҺж–°е»әж–Үд»¶жөӢиҜ•дёҖдёӢпјҢзңӢжҳҜдёҚжҳҜж–Үд»¶жІЎжңүиў«еҠ иҪҪеҲ°зј“еӯҳеҢәпјҢеҸ‘зҺ°з»“жһңе·Із»ҸеҠ иҪҪеҲ°зј“еҶІеҢәдәҶ

# ж·»еҠ  origin
